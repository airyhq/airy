`POST /messages.send`

Sends a message to a conversation from Google's Business Messages source and returns a payload.

Note that when you send a message from this source, you must specify a representative, the individual or automation that composed the message. 

Whatever is put on the `message` field will be forwarded "as-is" to the source's message endpoint.

**Sending a text message**

```json5
{
  "conversation_id": "a688d36c-a85e-44af-bc02-4248c2c97622",
  "message": {
    "text": "Hello World Agent!",
    "representative": {
      "displayName": "Hello World from Agent",
      "representativeType": "HUMAN"
    }
  }
}
```

**Sample response**

```json5
{
  "id": "{UUID}",
  "content": '{"text":"Hello"}',
  "state": "pending|failed|delivered",
  "sender_type": "{string/enum}",
  // See glossary
  "sent_at": "{string}",
  //'yyyy-MM-dd'T'HH:mm:ss.SSSZ' date in UTC form, to be localized by clients
  "source": "{String}",
  // one of the possible sources
  "metadata": {
    "sentFrom": "iPhone"
  }
  // metadata object of the message
}
```
